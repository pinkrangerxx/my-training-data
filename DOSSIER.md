## 3. Training Schedule & Framework

### Weekly Volume — Conservative Progression

Given bone stress injury history and recent load (2–3 runs/week through April 2026), frequency increases gradually.

| Phase | Weeks | Runs/Week | Logic |
|-------|-------|-----------|-------|
| Base | 1–4 | 4 | +1 from recent baseline |
| Build | 5–9 | 5 | Add one session as body adapts |
| Peak | 10–13 | 5–6 | Only if all readiness signals green |
| Taper | 14–16 | 4–5 | Progressive reduction |

**Weekly mileage increases:** ≤ 10% per week. Non-negotiable given bone stress history.

### Typical Week (Base Phase — Weeks 1–4)

| Day | Session | Notes |
|-----|---------|-------|
| Sunday | Long Run | Key session — weekend preferred |
| Monday | Rest or mobility | Full recovery |
| Tuesday | Easy Run (Z1–Z2) | Conversational pace only |
| Wednesday | Quality Session | Tempo or marathon-pace work |
| Thursday | Rest or easy cycling (Z1–Z2) | Optional cross-train |
| Friday | Rest | Prep for weekend |
| Saturday | Easy Run (Z1–Z2) | Short and easy |

### Recovery Protocol

**Recovery Triggers:**
- HRV ↓ > 20% → Z1–Z2 only, no quality work
- RHR ↑ ≥ 5 bpm → reduce volume 20%, no hard sessions
- Feel ≥ 4 → rest or deload
- Two+ triggers → complete rest

**⚠️ Bone Stress Injury Protocol — HIGH PRIORITY:**
- History: Bone stress injury Aug/Sep 2024 post-City2Surf. Fully recovered, physio-cleared.
- Mileage increases ≤ 10%/week
- Any shin, foot, or lower leg pain → stop immediately
- Treadmill substitution acceptable to manage impact load
- Peak long run capped at 32km unless signals are green 3+ weeks consecutively
